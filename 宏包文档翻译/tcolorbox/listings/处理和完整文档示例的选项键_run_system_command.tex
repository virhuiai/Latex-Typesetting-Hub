\begin{docTcbKey}[][doc new=2014-11-14]{run system command}{=\meta{system command}}{style, no default, initially unset}
Runs a \meta{system command}, if the document is compiled with the
|-shell-escape| permission.
The current listing file can be accessed as |\filename@area\filename@base\filename@ext|.
This \meta{system command} is added to \refKeyLe{/tcb/process code}.

如果文档使用|-shell-escape|权限编译，则运行\meta{系统命令}。当前的源码列表文件名为 |\filename@area\filename@base\filename@ext| 。这个\meta{系统命令}被添加到\refKeyLe{/tcb/process code}中。

\begin{dispExample}
\makeatletter
\providecommand\jobnameToRun{\filename@area\filename@base}
\makeatletter
\begin{tcblisting}{
title={virhuiai的例子：调用了pdfcrop},
listing side comment,
compilable listing,
run xelatex={-shell-escape},
run system command={%
pdfcrop\space\jobnameToRun.pdf\space\jobnameToRun.pdf},
pdf comment
}
\documentclass{minimal}
\begin{document}
\thispagestyle{empty}
test
\end{document}  
\end{tcblisting}
\end{dispExample}
\end{docTcbKey}
